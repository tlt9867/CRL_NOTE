Sarsa(on-policy)

  • Derive TD target
    • Q(st,at)=E[Rt+Q(St+1,At+1)]ytQ_\pi(s_t,a_t) = \mathbb E{[R_t+\gamma\cdot Q_\pi(S_{t+1},A_{t+1})]} \approx y_tQ(st,at)=E[Rt+Q(St+1,At+1)]yt, for all \pi
    • rt+Q(st+1,at+1)\approx r_t+\gamma\cdot Q_\pi(s_{t+1},a_{t+1})rt+Q(st+1,at+1)
    • encourage Q(st,at)Q_\pi(s_t,a_t)Q(st,at) to approximate yty_tyt
  • Tabular version
    • observe a transition (st,at,rt,st+1)(s_t,a_t,r_t,s_{t+1})(st,at,rt,st+1)
    • sample at+1(st+1)a_{t+1} \sim \pi(\cdot|s_{t+1})at+1(st+1), where \pi is the policy function
    • TD target: yt=rt+Q(st+1,at+1)y_t=r_t+\gamma\cdot Q_\pi(s_{t+1},a_{t+1})yt=rt+Q(st+1,at+1)
    • TD error: t=Q(st,at)yt\delta_t = Q_\pi(s_t,a_t )-y_tt=Q(st,at)yt

  • Value network version
    • Approximate Q(st,at)Q_\pi(s_t,a_t)Q(st,at) by the value network, q(s,a;)q(s,a;\omega)q(s,a;)
    • learn \omega
  • TD error and Gradient
    • TD target: yt=rt+q(st+1,at+1;)y_t = r_t +\gamma\cdot q(s_{t+1},a_{t+1};\omega)yt=rt+q(st+1,at+1;)

Q-learning(off-policy)

  • Sarsa VS Q-learning
    • Sarsa is for training action-value function, Q(s,a)Q_\pi(s,a)Q(s,a)
    • TD target: yt=rt+q(st+1,at+1;)y_t = r_t +\gamma\cdot q(s_{t+1},a_{t+1};\omega)yt=rt+q(st+1,at+1;)
    • Used Sarsa for updating value network(critic)
    • Q-learning is for training the optimal action-value function, Q(s,a)Q^\ast(s,a)Q(s,a)
    • TD target: yt=rt+maxaQ(st+1,a)y_t = r_t +\gamma\cdot \max\limits_a Q^\ast(s_{t+1},a)yt=rt+amaxQ(st+1,a)
    • Used Q-learning for updating DQN
  • Derive TD target
    • Q(st,at)=E[Rt+Q(St+1,At+1)]Q_\pi(s_t,a_t)= \mathbb E[R_t+\gamma\cdot Q_\pi(S_{t+1},A_{t+1})]Q(st,at)=E[Rt+Q(St+1,At+1)] for all \pi
    • if \pi is the optimal policy \pi^\ast, then
      Q(st,at)=E[Rt+Q(St+1,At+1)]Q_{\pi^\ast}(s_t,a_t)= \mathbb E[R_t+\gamma\cdot Q_{\pi^\ast}(S_{t+1},A_{t+1})]Q(st,at)=E[Rt+Q(St+1,At+1)]
    • QQ_{\pi^\ast}Q and QQ^\astQ both denote the optimal action-value function
    • identity: Q(st,at)=E[Rt+Q(St+1,At+1)]Q^\ast(s_t,a_t)= \mathbb E[R_t+\gamma\cdot Q^\ast(S_{t+1},A_{t+1})]Q(st,at)=E[Rt+Q(St+1,At+1)]
      • The action At+1=argmaxaQ(St+1,a)A_{t+1} = \argmax\limits_aQ^\ast(S_{t+1},a)At+1=aargmaxQ(St+1,a)
      • Q(st,at)=E[Rt+maxaQ(St+1,a)]Q^\ast(s_t,a_t)= \mathbb E[R_t+\gamma\cdot \max\limits_aQ^\ast(S_{t+1},a)]Q(st,at)=E[Rt+amaxQ(St+1,a)]
      • rt+maxaQ(st+1,a)\approx r_t+\gamma\cdot \max\limits_aQ^\ast(s_{t+1},a)rt+amaxQ(st+1,a) which is the TD target yty_tyt
    • Tabular version
      • observe a transition (st,at,rt,st+1)(s_t,a_t,r_t,s_{t+1})(st,at,rt,st+1)
      • TD target: yt=rt+maxaQ(st+1,a)y_t =r_t+\gamma\cdot \max\limits_aQ^\ast(s_{t+1},a)yt=rt+amaxQ(st+1,a)
      • TD error: t=Q(st,at)yt\delta_t = Q^\ast(s_t,a_t)-y_tt=Q(st,at)yt
      • Update: Q(st,at)Q(st,at)tQ^\ast(s_t,a_t)\leftarrow Q^\ast(s_t,a_t)-\alpha\cdot \delta_tQ(st,at)Q(st,at)t
    • DQN version
      • Approximate Q(st,at)Q^\ast(s_t,a_t)Q(st,at) by DQN, Q(s,a;)Q(s,a;\omega)Q(s,a;)
      • DQN controls the agent by:at=argmaxaQ(st,a;)a_t=\argmax\limits_aQ(s_t,a;\omega)at=aargmaxQ(st,a;)
      • learn \omega
      • Observe a transition (st,at,rt,st+1)(s_t,a_t,r_t,s_{t+1})(st,at,rt,st+1)
      • TD target: yt=rt+maxaQ(st+1,a;)y_t =r_t+\gamma\cdot \max\limits_aQ(s_{t+1},a;\omega)yt=rt+amaxQ(st+1,a;)
      • TD error: t=Q(st,at;)yt\delta_t = Q(s_t,a_t;\omega)-y_tt=Q(st,at;)yt

Actor-critic

value-based methods(critic)

  • Use neural net q(s,a;)q(s,a;\omega)q(s,a;) to approximate Q(s,a)Q_{\pi}(s,a)Q(s,a)
  • \omega is the trainable parameters of the neural net
  • input: state s and action a
  • approximate action-value (scalar)

Policy-based methods(actor)

  • Use neural net (as;)\pi(a|s;\theta)(as;) to approximate (as)\pi(a|s)(as)
  • \theta is the trainable parameters of the neural net
  • input: state s
  • output: probability distributions over the actions

Network training

  • observe state sts_tst
  • Randomly sample action ata_tat according to (st;t)\pi(\cdot|s_t;\theta_t)(st;t)
  • perform ata_tat and observe new state st+1s_{t+1}st+1 and reward rtr_trt
  • update value network q use TD
    • compute q(st,at;t)q(s_t,a_t;\omega_t)q(st,at;t) and q(st+1,at+1;t)q(s_{t+1},a_{t+1};\omega_t)q(st+1,at+1;t)
    • TD target: yt=rt+q(st+1,at+1;t)y_t = r_t +\gamma\cdot q(s_{t+1},a_{t+1};\omega_t)yt=rt+q(st+1,at+1;t)
    • Loss: L()=12[q(st,at;)yt]2L(\omega)=\frac{1}{2}[q(s_t,a_t;\omega)-y_t]^2L()=21[q(st,at;)yt]2
    • Gradient descent: t+1=tL()=t\omega_{t+1} = \omega_t-\alpha\cdot \frac{\partial L(\omega)}{\partial \omega}\Bigr|_{\substack{\omega=\omega_t}}t+1=tL()=t
  • update policy network \pi using policy gradient
    • Definition: State-value function approximated using neural networks
      • V(s;,)=a(as;)q(s,a;)V(s;\theta,\omega)=\sum_a\pi(a|s;\theta)\cdot q(s,a;\omega)V(s;,)=a(as;)q(s,a;)
    • policy gradient: derivative of V(st;,)V(s_t;\theta,\omega)V(st;,) w.r.t. \theta.
      • let g(a,)=log(as,)q(st,a;)g(a,\theta) = \frac{\partial log~{\pi(a|s,\theta)}}{\partial \theta}\cdot q(s_t,a;\omega)g(a,)=log(as,)q(st,a;)
      • V(st;,t)=EA[g(A,)]\frac{\partial V(s_t;\theta,\omega_t)}{\partial \theta} = \mathbb{E}_A{[g(A,\theta)]}V(st;,t)=EA[g(A,)]
    • random sampling: a(st;t)a \sim \pi(\cdot|s_t;\theta_t)a(st;t)
    • stochastic gradient ascent: t+1=t+g(a,t)\theta_{t+1}=\theta_t+\beta\cdot g(a,\theta_t)t+1=t+g(a,t)
  • summary of Algorithm
    1. Observe state sts_tst and randomly sample at(st;t)a_t\sim \pi(\cdot|s_t;\theta_t)at(st;t).
    2. Perform ata_tat and observe new state st+1s_{t+1}st+1 and reward rtr_trt
    3. Randomly sample a~t+1(st+1;t)\tilde{a}_{t+1} \sim \pi(\cdot|s_{t+1};\theta_t)a~t+1(st+1;t)
    4. Evaluate value network: q(st,at;t)q(s_t,a_t;\omega_t)q(st,at;t) and q(st+1,at+1;t)q(s_{t+1},a_{t+1};\omega_t)q(st+1,at+1;t)
    5. Compute TD error: t=qt(rt+qt+1)\delta_t = q_t - (r_t+\gamma\cdot q_{t+1})t=qt(rt+qt+1)
    6. Differentiate value network: dw,t=q(st,at;)=td_{w,t}=\frac{\partial q(s_t,a_t;\omega)}{\partial \omega}\bigr|_{\substack{\omega=\omega_t}}dw,t=q(st,at;)=t
    7. Update value network: t+1=ttdw,t\omega_{t+1} = \omega_t -\alpha\cdot \delta_t \cdot d_{w,t}t+1=ttdw,t
    8. Differentiate policy network: d,t=log(atst,)=td_{\theta,t}= \frac{\partial log~{\pi(a_t|s_t,\theta)}}{\partial \theta} \bigr|_{\substack \theta = \theta_t}d,t=log(atst,)=t
    9. Update policy network: t+1=t+td,t\theta_{t+1}=\theta_t+\beta\cdot \delta_t \cdot d_{\theta,t}t+1=t+td,t
